操作パターン１
─矢印キーベースの両手操作
右移動：右矢印キー
左移動：左矢印キー
ハードドロップ（即接地）：上矢印キー
右回転：Xキー
左回転：Zキー
ホールド：Cキー

操作パターン２
─WASDベースの両手操作
右移動：Dキー
左移動：Aキー
ハードドロップ（即接地）：Wキー, Spaceキー
右回転：Kキー
左回転：Jキー
ホールド：Hキー

操作パターン３
─WASDベースの片手操作
右移動：Dキー
左移動：Aキー
ハードドロップ（即接地）：Wキー, Spaceキー
右回転：Eキー
左回転：Qキー
ホールド：Fキー
